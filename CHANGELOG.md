# Changelog

All notable changes to this project will be documented in this file.

## [1.2] - 2026-01-05

### Fixed
- Fixed item depletion detection by adding usage statistics verification to prevent false triggers when items are manually moved or dropped from inventory

## [1.1] - 2025-12-20

### Added
- Sound notification option for shulker box refills with network handling
- Shulker box name filtering options for refill configuration
- Detailed mod description, author information, and contact links in fabric.mod.json

### Changed
- Enhanced build workflow with automated GitHub releases
- Improved artifact management excluding source jars from releases
